## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Florence100/frame-calculator
2. Navigate to the project directory 
   ```bash
   cd frame-calculator
4. Install the dependencies:
   ```bash
   npm install


## Usage

1. Start the project:
   ```bash
   npm run start

2. Open your browser and go to http://localhost:3000/
